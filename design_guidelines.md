# Ash3 - Design Guidelines

## Design Philosophy
A **cute, romantic, pastel interface** perfect for couples to track their shared bucket list. The entire UI should feel cozy, warm, and couple-friendly with playful animations and soft aesthetics.

## Brand Identity

### Logo
- Custom "Ash3" overlapping logo where "Ash" and "3" blend together creatively
- The "3" should integrate into the letters like a couple in a hug
- Soft curves and rounded letterforms
- Tiny sparkles or small heart accent
- Rose pink color (#FF80AB) as primary
- Adorable, minimal, and scales perfectly for navbar and mobile
- SVG format for crisp rendering

## Color Palette

**Primary Theme:** Rose Pink (#FF80AB)
- Use throughout for primary actions, accents, and highlights
- Soft gradients incorporating rose pink
- Pastel variations for backgrounds and secondary elements
- Light, romantic tones that create warmth

## Typography

**Font Family:** Poppins or Inter
- Clean, soft, modern appearance
- Excellent readability for couples using the app together
- Friendly and approachable letterforms

## Layout & Spacing

**Border Radius:** 20-32px consistently across all components
- Cards, buttons, inputs all use generous rounding
- Creates soft, friendly aesthetic

**Mobile-First Responsive:**
- Prioritize mobile experience
- Seamless scaling to tablet and desktop
- Touch-friendly interactive elements

## Interactive Elements

### Checkboxes
- Empty box by default
- Click to mark complete
- Heart pop animation on completion
- Confetti celebration when checked

### Buttons
- Bounce and scale animations using Framer Motion
- Soft shadows and gradients
- Heart icons where appropriate
- Playful hover states

### Animations (Framer Motion throughout)
- Page transitions with smooth fades/slides
- Button bounces on interaction
- Heart pop animations on completion
- Scale effects on hover
- Confetti burst when items completed
- Timeline animations in activity feed

## Page Layouts

### Main Checklist Page
- Starts completely empty - clean slate
- Prominent "Add Item" button with heart animation
- Each item shows: checkbox, text, optional description, optional priority label, optional target date
- Priority labels in cute pastel colors
- Soft card design for each item

### Completed Items Page
- Beautiful grid layout
- Shows completion dates
- Soft celebratory visuals
- Heart accents throughout

### Activity Feed Page
- Timeline-style vertical feed
- Cute icons for each action type (added, completed, deleted)
- Chronological display with timestamps
- Soft connecting lines between activities

### Analytics Page
- Animated statistic cards
- Total items added
- Items completed
- Progress percentage with visual representation (cute progress bars/circles)
- Heart-themed data visualization

### Settings Page
- Minimal, clean layout
- "Clear All Data" button with warning aesthetic
- Confirmation popup with soft rounded corners

## Visual Elements

**Heart Icons:** Throughout the interface for romantic touch
**Soft Gradients:** Subtle backgrounds and accent areas
**Sparkles/Stars:** Decorative elements for delight
**Progress Bars:** Cute, rounded with soft colors
**Confetti:** Canvas-based celebration on completions

## Component Design Principles

- Every interactive element should feel delightful
- Generous padding and spacing for comfortable interaction
- Soft shadows instead of hard borders
- Pastels over bright, harsh colors
- Rounded over sharp edges
- Animated over static where appropriate

## Images

**No hero images needed** - This is a functional web app focused on the checklist interface. The logo and pastel theme provide all the visual branding needed. The interface itself is the hero through cute animations and romantic design elements.
